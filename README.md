# Rodando o projeto
- composer install
- composer update
- php artisan key:generate
- Criar cópia do arquivo .env.example chamada de .env e configurar suas variavéis
- php artisan serve

## Instalando dependencias
- composer require laravel/breeze --dev
- php artisan migrate
- npm install
- npm run dev
- composer require barryvdh/laravel-dompdf
